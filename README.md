# Simple Flask API

## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/adzinzhalifunnas/Simple-Flask-API.git
   ```
2. Install requirements
   ```sh
   pip3 install -r requirements.txt
   ```
3. Run the app
   ```sh
   python3 app.py
   ```
4. Open the app in your browser
   ```sh
   http://127.0.0.1:5000
   ```
5. Enjoy!
